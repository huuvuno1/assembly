To mount a folder as a drive
>MOUNT [Drive-Letter] [Local-Directory]

Example
>MOUNT C ~/Document/assembly

Command to reset the mounted directory:
>rescan

Command to run code:
>RUN.bat filename

Example
>RUN.bat bai1
